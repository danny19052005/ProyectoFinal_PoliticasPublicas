# Carpeta de Código Fuente

## Propósito

Esta carpeta contiene el código fuente del proyecto, incluyendo módulos reutilizables, funciones auxiliares y utilidades que sustentan la funcionalidad de los agentes inteligentes.

## Tipos de Archivos

- **Archivos Python (.py)**: Módulos, clases y funciones auxiliares
- **Archivos de configuración**: settings.py, config.yaml
- **Archivos de documentación**: docstrings y comentarios en código
- **Archivos de pruebas**: test_*.py, conftest.py

## Documentación Requerida

Todos los módulos, funciones y componentes incorporados en esta carpeta deben estar correctamente documentados. Se recomienda incluir:

- **Docstrings**: Descripción de módulos, clases y funciones
- **Parámetros y retornos**: Tipos y significado de argumentos
- **Ejemplos de uso**: Cómo utilizar el código
- **Dependencias**: Módulos y librerías requeridas
- **Notas técnicas**: Consideraciones de implementación o limitaciones conocidas

Esto facilita el mantenimiento del código, la colaboración entre desarrolladores y la integración con otros módulos del proyecto.
