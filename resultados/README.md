# Carpeta de Resultados

## Propósito

Esta carpeta contiene todos los resultados, análisis y salidas generadas por los agentes del proyecto. Incluye reportes de síntesis, análisis temáticos, evaluaciones de políticas y conclusiones finales.

## Tipos de Archivos

- **Archivos Markdown (.md)**: Reportes y análisis en formato texto estructurado
- **Archivos JSON**: Resultados estructurados para procesamiento posterior
- **Archivos CSV**: Datos tabulares de resultados
- **Archivos PDF**: Reportes finales formalizados
- **Archivos HTML**: Reportes interactivos

## Documentación Requerida

Todos los resultados, análisis e información incorporados en esta carpeta deben estar correctamente documentados. Se recomienda incluir:

- **Título y resumen ejecutivo**: Qué contiene el resultado
- **Metodología**: Cómo se llegó a este resultado
- **Fuentes de datos**: Qué datos se utilizaron
- **Conclusiones principales**: Hallazgos clave
- **Fecha de generación**: Cuándo se produjo este resultado
- **Limitaciones**: Restricciones o consideraciones sobre los resultados

Esto asegura la transparencia, reproducibilidad y utilidad de los resultados generados.
