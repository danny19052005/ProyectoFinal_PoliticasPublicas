# Carpeta de Prompts

## Propósito

Esta carpeta almacena los prompts e instrucciones utilizadas para guiar a los agentes inteligentes en el análisis de políticas públicas. Contiene las definiciones de tareas, criterios de evaluación y directrices para cada agente.

## Tipos de Archivos

- **Archivos Markdown (.md)**: Prompts estructurados y directrices para agentes
- **Archivos de texto (.txt)**: Instrucciones simples o formatos específicos
- **Archivos JSON**: Prompts estructurados en formato de datos
- **Archivos YAML (.yml, .yaml)**: Configuración de prompts

## Documentación Requerida

Todos los prompts, instrucciones e información incorporados en esta carpeta deben estar correctamente documentados. Se recomienda incluir:

- **Nombre y descripción del prompt**: Qué tarea o análisis ejecuta
- **Agente objetivo**: A qué agente está dirigido este prompt
- **Versión**: Número de versión del prompt
- **Cambios recientes**: Qué se modificó en versiones anteriores
- **Notas de efectividad**: Observaciones sobre cómo funciona en la práctica

Esto facilita la mejora iterativa de los prompts y la reproducibilidad de los análisis.
