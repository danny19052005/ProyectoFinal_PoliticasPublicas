# Carpeta de Documentos

## Propósito

Esta carpeta almacena toda la documentación del proyecto, incluyendo análisis de políticas públicas, informes, marcos conceptuales y referencias teóricas que sustentan el análisis.

## Tipos de Archivos

- **Archivos Markdown (.md)**: Documentación en formato texto estructurado
- **Archivos PDF**: Reportes finales, papers y documentos formales
- **Archivos Word (.docx)**: Documentos editables de análisis
- **Archivos de texto (.txt)**: Notas y referencias

## Documentación Requerida

Todos los documentos, análisis e información incorporados en esta carpeta deben estar correctamente documentados. Se recomienda incluir:

- **Título y descripción**: Qué contiene el documento
- **Autor/es**: Quién produjo o contribuyó al documento
- **Fecha**: Cuándo se creó o actualizó
- **Referencias**: Fuentes citadas o documentos relacionados
- **Contexto**: En qué etapa del proyecto se utilizó este documento

Esto facilita el seguimiento de la evolución del análisis y la reproducibilidad del trabajo.
