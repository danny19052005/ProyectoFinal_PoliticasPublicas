# Fuentes oficiales F02-F06 (copias de trabajo)

Esta carpeta contiene copias de trabajo de fuentes oficiales usadas en el proyecto.

- Cada archivo debe conservar su código de fuente (F02, F03, F04, F05 o F06) al inicio del nombre.
- Los archivos originales descargados no deben modificarse.
- Una descarga exitosa no equivale a una verificación sustantiva del contenido.
- Toda cifra, afirmación o cita debe contrastarse con la fuente oficial original antes de su uso académico.
- Los metadatos de recopilación se registran en `resultados/recopilacion_fuentes_metadatos.json`.
