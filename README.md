# Análisis de implementación de la política pública de seguridad ciudadana y recuperación del espacio público en el Distrito Metropolitano de Quito

Este proyecto académico en Python tiene como objetivo construir una base estructurada para analizar la implementación de políticas públicas orientadas a la seguridad ciudadana y la recuperación del espacio público en el Distrito Metropolitano de Quito (DMQ).

## Propósito

El proyecto busca:
- sistematizar información relevante sobre la implementación de la política pública;
- organizar datos, documentos y hallazgos de manera ordenada;
- facilitar el desarrollo de análisis, visualizaciones y reportes académicos;
- servir como base para un dashboard o un flujo de trabajo reproducible.

## Estructura del proyecto

- datos/: archivos de entrada, bases de datos o datasets.
- documentos/: informes, referencias, artículos y documentos de apoyo.
- dashboard/: componentes o recursos para visualizaciones.
- agentes/: módulos o lógica de agentes de análisis.
- prompts/: plantillas o instrucciones para interacción con modelos o herramientas.
- resultados/: salidas, tablas, gráficos y reportes generados.
- src/: código fuente del proyecto.

## Herramientas sugeridas

El proyecto puede desarrollarse con Python, bibliotecas de análisis de datos y herramientas de visualización como:
- pandas
- numpy
- matplotlib
- seaborn
- streamlit
- jupyter

## Próximos pasos

1. Definir la pregunta de investigación y los indicadores clave.
2. Recopilar y organizar los datos relevantes.
3. Implementar scripts de procesamiento y análisis en src/.
4. Generar visualizaciones y resultados en dashboard/ y resultados/.

## Nota

Este repositorio está pensado como una base inicial para un trabajo académico, con una estructura profesional y escalable.
