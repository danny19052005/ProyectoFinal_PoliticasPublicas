# Carpeta de Datos

## Propósito

Esta carpeta almacena todos los datos utilizados por el proyecto de análisis de políticas públicas. Incluye datasets crudos, datos procesados e información necesaria para los análisis realizados por los agentes inteligentes.

## Tipos de Archivos

- **Archivos CSV**: Datasets procesados y limpios
- **Archivos JSON**: Datos estructurados e intercambios entre agentes
- **Archivos Excel (.xlsx)**: Datos tabulares complejos
- **Archivos de texto**: Fuentes de datos documentadas

## Documentación Requerida

Todos los datos, datasets e información incorporados en esta carpeta deben estar correctamente documentados. Se recomienda incluir:

- **Descripción del dataset**: Qué datos contiene y su origen
- **Diccionario de variables**: Explicación de cada columna o campo
- **Periodo de datos**: Fechas de cobertura del dataset
- **Fuentes**: Referencias a las instituciones o documentos de origen
- **Notas de calidad**: Limitaciones, valores faltantes o problemas conocidos

Esto asegura la reproducibilidad y transparencia del análisis.
